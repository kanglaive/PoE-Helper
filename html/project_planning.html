Project Planning

The project was initially intended for personal usage, as I found the commonly used AHK script unappealling, and was looking for a Java-based app to which I could add personal touches. Unfortunately, with no other application in sight, I pursued a project of my own.

In order to mimic the accessibility of AHK in terms of its ability to register hot keys/keyboard shortcuts across windows, I sought out jnativehook to allow my app the ability to register key presses outside of the app window. In Path of Exile, if a user presses Ctrl + C while hovering over an item, that item's stats are copied as a String into the system clipboard. The primary purpose of the item parsing feature is to take this item string from the clipboard and parse it into a Java object, in which the app can appropriately determine that item's mods, and showing said properties to the user.
